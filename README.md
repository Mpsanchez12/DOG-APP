## Descripción
Dog App es una aplicación web dinámica que combina la ternura de los perritos con el aprendizaje de código. Permite a los usuarios explorar una galería de imágenes de perros aleatorios, desbloquear un tip de programación al ver sus detalles, y guardar sus favoritos en una colección personal almacenada localmente en el navegador.
## API utilizada
- [Dog API](https://dog.ceo/dog-api/)

## Cómo ejecutar
1. Descargar el proyecto.
2. Abrir `index.html` en tu navegador.
3. Disfrutar de la app.
